## 🚀 **Quick Start for Your Subordinate**

### **Option 1: Electron GUI Installer (Recommended)**
```bash

cd SEFIROT-COMPLETE-INSTALLER

# Install dependencies
npm install

# Run installer
npm start
```

### **Option 2: Command Line Installation**
